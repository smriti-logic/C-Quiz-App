# C Quiz Application

This is a simple console-based quiz application written in C. The project asks multiple-choice questions and calculates user's score based on correct answers.

## Features 
- Multiple-Choice questions
- Score calculations
- Simple and interactive console output

 ## Technologies Used
 - C Programming Language
 - Git and GitHub

 ## How to Run The Project
 1. Download the project from GitHub :
    ```Bash
    git clone
    https://github.com/smriti-logic/C-Quiz-App.git
 2. Open the project folder :
    Bash
    cd C-Quiz-App
 4. Compile the Program:
    ```Bash
    gcc quizApp.c -o quiz
    ```
 5. Run the program:
    ./quiz
